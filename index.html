Work

I am currently working at Oracle, following their acquisition of Ksplice, where I had previous worked. I continue to work on the Ksplice technology and product for updating the Linux kernel without rebooting.

livegrep

As a side project, I've put together livegrep.com, a site which allows for realtime regex searching of the Linux kernel as-you-type, using Russ Cox's RE2 regex library, and a custom indexing backend.

Security

In my free time, I do some work on security for various open-source projects, including the Linux kernel and KVM. I presented at Black Hat USA 2011 about some work I have done on KVM. My slides and code for that talk are now available online.
